Start by adding Participants

  • 1)Add the trip name in the place holder
  • 2)Type participant name and click on the add button to add the participant.
  • 3)Click on the remove button beside participant to remove.
  • 4)Click next to save the participants added.

Creating an event

  • 1)Add the event name in the place holder
  • 2)Type the total amount spent for the event.
  • 3)You can select the participants who paid for the event.
  • 4)Click next to go to the page of who should pay back.

Tools

  • 1)Use the create event button to start by creating an event
  • 2)After selecting an event use edit event button to modify the info of that event.
  • 3)Use edit participants to add/remove participants.
  • 4)After adding all required events click show results for final results.

Events List

  • 1)Click on a particular event to get its stats in the workspace.
  • 2)You can use the search filter to find a particular event.
